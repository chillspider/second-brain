# AI Second Brain — Starter Vault

Welcome! This is your starter vault for the AI Second Brain system.

## ⚠️ Before you extract: install The Unarchiver

**Do NOT double-click the ZIP yet.** macOS's built-in unzipper strips hidden folders that the vault needs to work. You need a free app called **The Unarchiver** to extract it correctly.

1. **Download The Unarchiver** from the Mac App Store (it's free): [The Unarchiver](https://apps.apple.com/us/app/the-unarchiver/id425424353)
2. Once installed, **right-click** the ZIP file → **Open With** → **The Unarchiver**
3. Choose where to extract it (Desktop is fine)

That's it — The Unarchiver handles everything correctly.

---

## Getting Started

1. **Extract the ZIP** using The Unarchiver (see above)
2. **Open the extracted folder in Obsidian** — File → Open Vault → select the `Second Brain Starter` folder
3. **Trust the vault** — Obsidian will ask if you trust it. Click "Trust and Enable Plugins".
4. **Open Claude Cowork** and select this folder as your workspace
5. **Install the skills** — open the `Skills/` folder and double-click each `.skill` file to install them in Cowork
6. **Say "set up my second brain"** — Claude will walk you through the rest

That's it. The system does the rest.
